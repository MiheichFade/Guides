# Как пользоваться VirtualBox на ОС Windows  


## Общие сведения  


**VirtualBox** — это бесплатное программное обеспечение для виртуализации, 
позволяющее запускать на одном компьютере несколько операционных систем одновременно. 
**VirtualBox** создает <span style="color: #6a8ed0;" title="виртуальная машина">ВМ</span> — изолированные виртуальные среды, 
которые действуют как отдельные "мини-компьютеры" внутри основного компьютера.  

> Виртуализация предполагает использование двух видов операционных систем:  
> 
> - Хостовая <span style="color: #6a8ed0;" title="виртуальная машина">ОС</span> — это основная операционная система, 
> установленная на физическом компьютере, которая управляет **VirtualBox** и виртуальными машинами.  
> 
> - Гостевая ОС — это операционная система, установленная на виртуальной машине, 
> которая работает независимо от хостовой ОС и полностью изолирована от нее.  

Использование **VirtualBox** предоставляет следующие возможности:  

- Экономия ресурсов и удобство настройки.  

    ВМ позволяют работать с несколькими ОС на одном устройстве 
    без необходимости в дополнительном оборудовании. ВМ легко настраиваются и 
    позволяют пользователям быстро переключаться между системами.  

- Изоляция процессов.  

    Программы и процессы, выполняемые на ВМ, полностью изолированы от основной системы, 
    что снижает риск воздействия на хостовую ОС.  

- Тестирование и разработка.  

    ВМ отлично подходят для тестирования ПО или изменений в системных настройках 
    без риска повлиять на работу основной системы.

## Установка VirtualBox


Для установки **VIrtualBox** необходимо выполнить следующие шаги:  

1. Перейти на [официальный сайт разработчика](https://www.virtualbox.org/).  

1. Нажать на кнопку <span style="border:2px solid #555; border-radius:5px; width: 200px; text-align:center; padding:3px;">Download</span>.  

1. На открывшейся странице в разделе **VirtualBox Platform Packages** нажать на ссылку [**Windows hosts**](https://download.virtualbox.org/virtualbox/7.1.4/VirtualBox-7.1.4-165100-Win.exe) для начала загрузки установочного `.exe`-файла.  

    > Имя загружаемого файла: `VirtualBox-{version}-Win.exe`.  

1. Дождаться завершения загрузки.  

1. Открыть загруженный `.exe`-файл.  

1. На этапе **Welcome to the Oracle VirtualBox {version} Setup Wizard** нажать кнопку 
<span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Next ></span>.  

1. На этапе **End-User License Agreement** ознакомьтесь с лицензионным соглашением. 
Для его подтверждения и перехода к следующему этапу необходимо выбрать **I accept the terms in the License Agreement** и 
нажать кнопку 
<span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Next ></span>.  

1. На этапе **Custom Setup** имеется возможность выбора дополнительных компонентов, доступных к установке:  

    - VirtualBox Application

        > Основная программа, управляющая ВМ и обеспечивающая доступ ко всем функциям **VirtualBox**.  
        > 
        > Обязательна к установке.  

        - VirtualBox USB Support

            > Позволяет подключать USB-устройства (например, flash-накопители, принтеры или камеры) напрямую к ВМ.  

        - VirtualBox Networking

            > Сетевые компоненты, обеспечивающие ВМ подключением к сети. Содержит два следующих компонента:  

            - VirtualBox Bridged Networking

                > Позволяет ВМ напрямую подключаться к сети как отдельное физическое устройство.  
                > 
                > ВМ получает IP-адрес из той же сети, что и хост, и взаимодействует с другими устройствами 
                > в локальной сети так же, как если бы она была отдельным устройством.  

            - VirtualBox Host-Only Networking

                > Создает изолированную сеть, доступную только для хоста и ВМ.  
                > 
                > Этот компонент полезен для настройки локального взаимодействия между хостом и ВМ 
                > без подключения к внешней сети или сети Интернет.  
        
        - VirtualBox Python Support

            > Компонент, добавляющий поддержку скриптов на языке Python для автоматизации действий с ВМ.  
            > 
            > Этот компонент полезен для разработчиков или системных администраторов, 
            > которым необходимо выполнять автоматизированные задачи (например, создание или удаление ВМ, 
            > настройка параметров) с использованием Python API VirtualBox.  

    Для исключения компонента из списка устанавливаемых необходимо нажать на ![img](./_assets/component.jpeg) 
    и выбрать **× Entire feature will be unavailable**.  

    Рекомедуется оставить все установленные значения по умолчанию без изменений.  

    Также на этапе **Custom Setup** имеется возможность выбрать путь установки **VirtualBox**.  

    Для его изменения необходимо нажать на кнопку 
    <span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Browse</span>,
    в открывшемся окне выбрать директорию, предпочитаемую для установки **VirtualBox** и нажать на кнопку 
    <span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">ОК</span> 
    (или <span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Cancel</span> для отмены 
    изменения директории по умолчанию).  

    Для перехода на следующий этап установки необходимо нажать на кнопку 
    <span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Next ></span>.  

1. Если на этапе выбора установки дополнительных компонентов был выбран **VirtualBox Bridged Networking**, 
установщик предупредит о временном отключении устройства от сети для конфигурации интерфейсов.  

    Необходимо согласиться с продолжением установки, нажав на кнопку 
    <span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Yes</span>.  

1. Если на этапе выбора установки дополнительных компонентов был выбран **VirtualBox Python Support**, 
установщик проверит наличие зависимостей **Python Core** и **win32api** и, в случае их отсутствия, 
предупредит о необходимости самостоятельной настройки указанных компонентов.  

    > Подробнее о средствах разработки описано в документации **VirtualBox SDK**, 
    > доступной в [соответствующем разделе на официальном сайте разработчика](http://download.virtualbox.org/virtualbox/SDKRef.pdf).  

    Необходимо согласиться с продолжением установки, нажав на кнопку 
    <span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Yes</span>.  

1. На следующем этапе установки предлагается:  

    - Create start menu entries

        > Добавить ярлыки **VirtualBox** в меню **Пуск**.  

    - Create a shortcut on the desktop

        > Создать ярлык **VirtualBox** на рабочем столе.  

    - Create a shortcut in the Quick Launch Bar

        > Создать ярлык **VirtualBox** на панели быстрого запуска.  

    - Register file assotiation

        > Зарегистрировать файлы формата `.vbox` (конфигурационные файлы виртуальных машин **VirtualBox**) в системе, 
        > чтобы они автоматически запускались в **VirtualBox** при открытии.  

    Рекомедуется оставить все установленные значения по умолчанию без изменений.  

    Для перехода на следующий этап установки необходимо нажать на кнопку 
    <span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Next ></span>.  

1. На этапе **Ready to Install** необходимо нажать на кнопку 
<span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Install</span> 
для запуска процесса установки или 
<span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Back</span> 
для возврата к настройкам.  

1. Дождать завершения установки.  

1. В открывшемся окне включить опцию **Start Oracle VirtualBox {version} after installation** и нажать на кнопку 
<span style="border:2px solid #555; border-radius:0px; width: 200px; text-align:center; padding:3px;">Finish</span>.  

1. Наблюдать открывшееся окно с интерфейсом **VirtualBox**.
